# Sentiment Analysis Of Movies Reviews
## Overview

This project applies Natural Language Processing (NLP) to analyze the sentiment of movie reviews.
It predicts whether a review is positive or negative, helping businesses, platforms, or researchers understand audience opinions.

## Tech Stack

- Programming: Python

- Libraries: NLTK, scikit-learn, Pandas, NumPy

- Machine Learning Models: Logistic Regression, Naive Bayes

- Data: Sentiment polarity datasets

## Key Features

- Preprocesses text data (tokenization, stopword removal, and cleaning)

- Trains models to classify reviews as positive or negative

- Evaluates model performance using metrics like accuracy, precision, and recall

## Approach

- Collected and cleaned sentiment polarity datasets

- Performed text preprocessing and feature extraction

- Trained and evaluated NLP classification models

- Visualized results to understand sentiment distribution

## Use Case

- Movie streaming platforms can analyze user reviews for insights

- Film studios can gauge audience response before or after release

- Useful for research in text mining and NLP

## Why This Project Matters

Understanding audience sentiment is critical in the entertainment industry and digital platforms.
This project demonstrates practical NLP skills applied to real-world data.
